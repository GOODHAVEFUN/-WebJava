메소드

	클래스안에 선언되어있으면 메소드
	
메소드 정의
	메소드 밖에서 정의
	
	리턴타입 메소드명(자료형 매개변수)
	{
		실행할 문장;
		return 리턴값;
	}
	
메소드 정의 순서
	1. 기능을 생각한다
		({1, 2, 3, 4, 5}배열의 합을 더하는 기능)
		
	2. 리턴타입을  생각한다(void)
		void
	3. 기능에 맞는 메소드명을 작성한다
		void totalArray(){}
		
	4. 매개변수를 생각한다
		void totalArray(int[] arr){}
		
	5. 실행할 문장을 생각한다
		void totalArray(int[] arr){
			int sum = 0; //합을 저장할 변수
			for(int i : arr){
				sum+=i;
			}	
		}
		
	6. 리턴값을 생각한다
		int totalArray(int[] arr){
			int sum = 0; //합을 저장할 변수
			for(int i : arr){
				sum+=i;
			}
			return sum;	
		}
